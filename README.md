# Full Stack REST API Skeleton

The full-stack rest api skeleton application on Javascript with MEVN.

## Requirements

- express ^4.17
- vue ^2.5

## Quick start

```bash
$ cd client && npm run install
$ npm install
$ cd client && npm run build
```
